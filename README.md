
# flutter_plugin_bbd_application

  Flutter plugin to receive data available to any Blackberry Dynamics Application.

## Getting Started

> The `FlutterPluginBbdApplication` object provides access to information that is globally available to any BlackBerry Dynamics Application.

> __Note:__ `flutter_plugin_bbd_base` is dependent on 
> * `BBD Flutter Base plugin`: [install and configure](https://github.com/oli06/flutter_plugin_bbd_base)

## API reference

### Available functions

* `FlutterPluginBbdApplication.getApplicationConfig` 
* `FlutterPluginBbdApplication.getVersion`  
* `FlutterPluginBbdApplication.showPreferenceUI`  

For further information see the [example project](./example)
